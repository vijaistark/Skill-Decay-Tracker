# Skill Decay Tracker

A simple project to track skill improvement and skill decay over time.

## Features
- Add skills
- Track practice sessions
- Monitor consistency
- View decay percentage
- Analyze progress

## Tech Stack
- Python
- Flask
- SQLite

## Installation

```bash
git clone https://github.com/vijaistark/Skill-Decay-Tracker.git
cd Skill-Decay-Tracker
pip install -r requirements.txt
python app.py
```
